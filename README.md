
# Aplicação back-end de uma API utilizada nas aulas de Web do IFPI Campus Picos

### Prof. Jesiel Viana


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm start
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
